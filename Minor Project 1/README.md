# Road Accident Prediction in India

## Project Overview

This project focuses on analyzing road accident data from India and building a machine learning model to predict accident severity. The dataset contains various factors related to accidents such as weather conditions, road conditions, vehicle information, casualties, and accident severity.

The main objective is to identify patterns that contribute to road accidents and develop predictive models that can assist in improving road safety and decision-making.

## Dataset Information

* Dataset Name: Accident Prediction India
* Records: Approximately 3000 accident records
* Features: 22 columns
* File Format: CSV

### Important Features

* Accident Severity
* Number of Casualties
* Number of Vehicles
* Weather Conditions
* Road Surface Conditions
* Light Conditions
* Speed Limit
* Driver License Status
* Traffic Control Presence
* Vehicle Type
* Day of Week

## Project Workflow

### 1. Data Collection

The accident dataset was collected and imported into Python using Pandas.

### 2. Data Preprocessing

* Missing value detection
* Handling missing values using Mode Imputation
* Duplicate value checking
* Data type conversion
* Feature encoding

### 3. Exploratory Data Analysis (EDA)

The following analyses were performed:

* Accident Severity Distribution
* Casualties Analysis
* Weather Condition Analysis
* Road Surface Analysis
* Speed Limit Analysis
* Correlation Analysis
* Feature Relationship Visualization

### 4. Data Visualization

Various visualization techniques were used:

* Bar Charts

* Swarm Plot

## Key Findings

1. Accident severity is influenced by weather conditions and road surface conditions.
2. Higher numbers of casualties are generally associated with severe accidents.
3. Speed limits show a significant relationship with accident severity.
4. Proper handling of missing values improves model performance.
5. Data visualization helps identify important accident patterns.
